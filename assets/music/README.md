# Background Music

Place your audio file here:

**Expected filename:** `naan-un-azhaginile-bgm.mp3`

The website will automatically play "Naan Un Azhaginile" BGM by A.R. Rahman as background music.

The music will start playing when the user first interacts with the page (clicks anywhere).
A floating music toggle button (♪) appears in the bottom-right corner.

## How to add:
1. Get the MP3 file of "Naan Un Azhaginile" BGM
2. Name it `naan-un-azhaginile-bgm.mp3`
3. Place it in this folder (`assets/music/`)
4. The website will detect and play it automatically
