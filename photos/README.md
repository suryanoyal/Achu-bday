# Photo Gallery

Add your photos to this folder, then update `data/photos.json` with entries like:

```json
{
  "src": "photos/your-photo-name.jpg",
  "caption": "Description of the memory",
  "year": "2024"
}
```

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

Recommended: Keep images under 2MB each for fast loading.
